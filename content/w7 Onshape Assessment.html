w7 Onshape Assessment

Q: What is the difference between a fixed and a moveable assembly in Onshape ( Onshape 中固定組件和可移動組件之間有什麼區別 ) ? 

A: A Fixed assembly is created using the Top-Down Design method in a Part Studio and is static, while a Moveable assembly is built in an Assembly tab using Mates and can be used to visualize motion ( 固定裝配使用零件中的自頂向下設計方法創建，並且是靜態的，而可移動裝配合使用在裝配選項卡中構建，可用於可視化運動 ) .
